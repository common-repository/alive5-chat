== Description ==
Mission Control for Messaging. Live Chat, Texting, Chatbots, Facebook Messenger. Amplify your sales and support with Alive5.

== Note == 
This plugin uses a script from https://alive5.com. Primarily this script is being used to produce a web chat window that lets website visitors chat in real time with sales and support people. 
Secondarily it helps maintain the chat session as the visitor goes from one web page to another page. The script does not collect any other user data.
For more details please visit https://www.alive5.com/terms-of-service and https://www.alive5.com/privacy-policy
